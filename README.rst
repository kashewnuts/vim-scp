About
-----

Vim syntax for Keyence Script Launguage(BT Series)

Installation
------------

dein.vim
~~~~~~~~

Set up `dein.vim <https://github.com/Shougo/dein.vim>`_ then add ``call dein#add('kashewnuts/vim-scp')`` to your vimrc and run ``:call dein#update()`` from a fresh vim.
